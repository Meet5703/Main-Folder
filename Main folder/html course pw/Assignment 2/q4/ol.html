Orderd List Of HTML Tags

  1. HTML :- represents the root of an HTML document.
  2. hr :- element is most often displayed as a horizontal rule that is used to separate content (or define a change) in an HTML page.
  3. div :- defines a division or a section in an HTML document.
  4. Body :- defines the document's body.
  5. p :- defines a paragraph.
  6. ol :- defines an ordered list. An ordered list can be numerical or alphabetical.
  7. a :- defines a hyperlink, which is used to link from one page to another.
  8. b :- specifies bold text without any extra importance.
